# YHChartView
Charts that support zoom, drag, and rotation, including vertical bar, horizontal bar, and line chart.

## Cocoapods support
Podfile:

```
pod 'YHChartView'
```

## Manual install
1.clone or download respository.


2.copy 'YHChartView' to your project.
